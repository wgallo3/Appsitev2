SAFECompute - Access the CPU of Every Computer

Hypothetical Logo
Click image to zoom

The first version of the SAFE Network will not have distributed computation built into the core, but it is part of the MaidSafe development team's plan to build it into the next iteration of the Network. SAFE storage must come first.

History

Decentralized computation on the SAFE Network is inevitable, and current code is being written with this in mind as a future update. While it is not being worked on currently due to the greater importance of releasing a global version of the network to humanity, it is definitely planned for the next iteration of the SAFE Network, and opens countless exciting possibilities.

Features

Once the SAFE Network is capable of distributed computation, it will allow anybody in the world to instantly access all of the computing power of every device connected to the internet. If that is not amazing enough, it will also let everyone earn money at any moment, by renting out their device's computing power to the Network, for others to use. This could be used for large scale medical research / testing, the solving of previously impossible physics / mathematics problems, or the processing of impossibly large data sets.

Launch Date

It was expected that the network would launch at the end of last year, (and this app soon after,) however an amazing new coding language came out called "Rust" that turned out to be perfect for the SAFE Network to be created with. So, the team decided to restart the whole network from scratch in Rust. This means we will have to wait a few more months to see the network or this app. Follow it on Github!
