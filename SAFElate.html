SAFELate - Distributed Language Translation

Skype's Recent Innovation
Double-click for Fullscreen

More of a bare essential than an exciting new technology, SAFElate will translate data without any third party, and it's open-source design could be the translation backbone for all sorts of apps. News apps that report events in local languages, or instant messaging that allows you to chat with anyone on the planet without intermediaries, etc.

History

First explicitly mentioned on the MWApps thread, the need for translation services on the SAFE Network is quite important. Given the wide variety of apps that will need to integrate this functionality, the first open-source design for translation available on SAFE will make extreme amounts of money. With so many different apps using the basic framework, the coder of SAFElate translation tools for SAFE will earn potentially millions of dollars worth of SafeCoin around the clock.

Features

Basic requirements for such an app would include an open-source design with easy integration into other apps, support for all known languages, and perhaps voice recognition / playback. To take it further, the ability to crowdsource voice recordings and exact translations would give this app an unstoppable edge against any potential competitors, and secure its widespread adoption and financial success.

Launch Date

It was expected that the network would launch at the end of last year, (and this app soon after,) however an amazing new coding language came out called "Rust" that turned out to be perfect for the SAFE Network to be created with. So, the team decided to restart the whole network from scratch in Rust. This means we will have to wait a few more months to see the network or this app. Follow it on Github!
